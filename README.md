📌 Overview

Counter App is a modern and responsive React.js application built to practice the useState Hook. It allows users to increment, decrement, and reset the counter value through an interactive and user-friendly interface. Counter applications are commonly used to learn React state management and event handling concepts.

✨ Features
➕ Increment Counter
➖ Decrement Counter
🔄 Reset Counter
⚛️ Built with React.js
🎯 State Management using useState Hook
📱 Fully Responsive Design
🎨 Modern and Clean UI
⚡ Fast and Lightweight
